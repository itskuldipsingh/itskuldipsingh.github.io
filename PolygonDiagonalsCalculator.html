सूत्र

किसी बहुभुज के लिए विकर्णों की संख्या = (n - 3)*(n / 2)
जहाँ n = भुजाओं की संख्या है.

किसी बहुभुज के लिए विकर्णों की संख्या को निम्नलिखित सूत्र से ज्ञात कर सकते है 

विकर्ण = (n - 3)*(n / 2)

जहाँ n = भुजाओं की संख्या है। 

उदाहरण :

त्रिभुज में विकर्णों की संख्या 

n = 3

विकर्णों की संख्या = (n - 3)*(n / 2)

= (3 - 3) * (3 / 2) 

= 0


चतुर्भुज में विकर्णों की संख्या 

n = 4

विकर्णों की संख्या = (n - 3)*(n / 2)

= (4 - 3) * (4 / 2) 

= 2


पंचभुज में विकर्णों की संख्या

n = 5

विकर्णों की संख्या = (n - 3)*(n / 2)

= (5 - 3) * (5 / 2) 

= 5
